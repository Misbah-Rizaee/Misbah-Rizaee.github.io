My Work

If you are interested in seeing some of the projects I have worked on, feel free to check out my GitHub account, where you will find the source code for each one.

The Potential of LDA on Social Media

In my research project, I investigated the potential of topic modelling on social media to support the identification of refugee needs in near real-time. Using Twitter's API, I collected and pre-processed large volumes of data with NLP libraries, applying Latent Dirichlet Allocation (LDA) to identify latent themes. This work highlighted how social media insights can be used to identify areas requiring the most assistance.

Autonomous IoT-based Sorting Machine

This project involved prototyping an IoT-based autonomous sorting machine with a rotating conveyor belt controlled by an ESP32-WROOM-32 microcontroller. It used sensors and cloud communication for real-time package sorting. This project showcased my expertise in IoT, automation, and cloud-based data transmission.

Predicting Fire Brigade Notional Price

This project involved working with London Fire Brigade's dataset to predict the notional price of fire services using machine learning. By analyzing linear data patterns, we developed an accurate linear regression model to predict pricing. This work demonstrated my ability to analyze large datasets and apply ML techniques effectively.
